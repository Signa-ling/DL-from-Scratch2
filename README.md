# DL-from-Scratch2

## 概要

- [ゼロから作るDeepLearning②](https://www.oreilly.co.jp/books/9784873118369/)のお勉強リポジトリ
- jupyter notebookで各章のざっくりとした説明込で記載
  - 一部コード(章をまたいで使用されるもの)に関しては[本家リポジトリ](https://github.com/oreilly-japan/deep-learning-from-scratch-2)より抜粋

## 内容

- Chapter〇〇.ipynb
  - 各章のコードとアバウトな説明
- dataset
  - 章内で用いられる外部からのデータのDLコード(本家リポジトリより抜粋)
- common
  - 複数の章で使用されるコード(本家リポジトリより抜粋)
- requirements.txt
  - 本リポジトリで使用されるライブラリを記載
